# 航空总线

航空总线是航空电子系统中**设备间数据传输的核心协议簇**，需满足高可靠性、实时性、抗干扰性和容错性（航空领域核心诉求）。



| 总线类型         | 核心应用场景                     | 传输速率        | 拓扑结构           | 帧格式 / 编码特点                   | 关键优势                         | 工程关注点（数据处理 / 建模）                                |
| ---------------- | -------------------------------- | --------------- | ------------------ | ----------------------------------- | -------------------------------- | ------------------------------------------------------------ |
| ARINC 429        | 座舱显示、导航、发动机参数       | 100/12.5 kbps   | 点对点（单向）     | 32 位固定帧（8 位标签 + 24 位数据） | 抗干扰强、协议简单、成熟稳定     | 双极性归零码（BRZ）解码、大端字节序、标签映射（Label Mapping） |
| MIL-STD-1553B    | 飞控、航电核心系统（关键数据）   | 1 Mbps          | 双冗余总线型       | 命令帧 / 数据帧 / 状态帧（16 位字） | 实时性高、容错性强、支持多终端   | 时分多路复用（TDM）、BC/RT 模式、冗余设计、CRC 校验          |
| CANaerospace     | 非关键系统（照明、空调、传感器） | 125 kbps~1 Mbps | 总线型             | 扩展 CAN 帧（29 位 ID+8 字节数据）  | 成本低、兼容性好、轻量化         | CAN ID 解析、数据段打包 / 解包、位时序配置                   |
| AFDX (ARINC 664) | 高速数据传输（航电网络骨干）     | 100 Mbps/1 Gbps | 星型（以太网扩展） | 虚拟链路（VL）+ UDP/IP 封装         | 带宽大、确定性传输、支持多数据流 | VL 调度配置、帧过滤、IP/UDP 头解析、延迟控制                 |
| ARINC 629        | 大型客机分布式系统               | 2 Mbps          | 总线型（双向）     | 可变长帧（最多 256 字节）           | 支持多终端并发、容错性强         | 冲突检测与避让、帧同步、循环冗余校验（CRC）                  |
